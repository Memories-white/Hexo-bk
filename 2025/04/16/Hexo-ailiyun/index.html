本地主机配置

安装 Node.js

浏览器进入 NodeJS 官网，点击下载 Node.js(LTS 版本)安装包。验证 nodeJS 是否安装成功：按下 win+r 进入 cmd 窗口，输入 node -v，出现版本号即为安装成功。
如果不成功，就需要去环境变量中添加 node.js 的安装路径，或者重新安装 node.js。
